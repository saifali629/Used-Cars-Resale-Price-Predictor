# Used Car Price Prediction (Machine Learning)

This project predicts the resale price of used cars using supervised machine learning models.

## Models Used
- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor

## Workflow
1. Data cleaning & preprocessing
2. One-hot encoding of categorical features
3. Train-test split
4. Model training & evaluation
5. Best model selection based on RMSE
6. Model saving and predictions
